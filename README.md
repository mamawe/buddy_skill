# buddy_skill

我的 WorkBuddy Skills 开源集合。

---

## 可用 Skills

### seed-more

复刻得到大脑「发芽」功能——从知识文档提取种子，跨域关联，生成 Aha 金句报告。优先关联用户 ima 知识库，搜不到才补互联网。

**安装**：
```bash
npx skills add mamawe/buddy_skill --skill seed-more
```

**触发词**：`发芽` `germinate` `sprout` `得到发芽` `知识发芽` `跨域关联` `知识连接`

---

### NoFluff

拒绝阅读 AI 水文——评估文章各章节的实际阅读价值，识别 AI 生成的低质量内容（AI slop），输出章节热力图、阅读建议与理由。

**安装**：
```bash
npx skills add mamawe/buddy_skill --skill NoFluff
```

**触发词**：`这篇文章是不是水文` `评估这篇文章的阅读价值` `这篇文章值不值得读` `NoFluff 评估一下`
