# seed-more

复刻得到大脑「发芽」功能——从知识文档提取种子，跨域关联，生成 Aha 金句报告。优先关联用户 ima 知识库，搜不到才补互联网。

## 安装

```bash
npx skills add mamawe/buddy_skill --skill seed-more
```

## 适用 Agent

- WorkBuddy
- Claude Code
- Cursor
- 任何支持 [Agent Skills](https://agentskills.io) 规范的 Agent

## 功能说明

不是压缩，是**发散**。把文档中的"种子"和用户已有知识体系建立跨域连接，生成一篇**发芽报告**。

**四步流水线**：

1. **提取种子** —— 从文档中识别有理论延伸潜力的知识点
2. **跨域关联** —— 优先搜 ima 知识库，补互联网
3. **Aha 提炼** —— 找到张力点，用悖论式金句引爆
4. **金句回响** —— 原文金句 + 跨域思想回响

## 触发词

`发芽` `germinate` `sprout` `得到发芽` `知识发芽` `跨域关联` `知识连接`

## 与得到大脑 original 的差异

| 维度 | 得到大脑「发芽」 | seed-more |
|------|----------------|----------|
| 知识来源 | 得到知识库（固定） | 用户自己的 ima 知识库（个性化）|
| 输出格式 | 固定报告 | Markdown，可自定义 |
| 跨域范围 | 预设知识图谱 | 取决于用户知识库 + 互联网 |
| 可复用性 | 一次性 | Skill，可反复使用 |
| 透明度 | 黑盒 | 四步流水线完全透明 |

**核心优势**：每个人的知识库不同，所以每个人的发芽报告都不同。这才是真正的个性化。
